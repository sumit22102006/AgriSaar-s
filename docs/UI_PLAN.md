# UI Plan

## Pages Overview

### 1. Home (/)
- Hero section with gradient green background
- Floating crop/farm emojis
- Stats cards (AI Modules, Crops, Schemes)
- Feature grid with 7 service cards
- CTA section

### 2. Soil Input (/soil-input)
- Tab switch: Manual Entry / Upload Report
- Voice input button (Hindi)
- Form with N, P, K, pH, OC, Location, Crop
- Info section (where to get soil report)

### 3. Analysis Dashboard (/analysis)
- Soil health score gauge (0-100)
- Nutrient progress bars
- AI analysis text
- Fertilizer table
- Crop recommendations
- Quick links to Weather/Market/Schemes

### 4. Crop Recommendations (/crops)
- Soil form input
- AI recommendation display
- Crop cards with ranking

### 5. Fertilizer Plan (/fertilizer)
- Form input (soil + crop)
- Fertilizer table
- AI detailed plan
- Safety warnings

### 6. Weather (/weather)
- Location search
- Current weather card (gradient)
- 8-slot forecast grid
- AI farming advisory

### 7. Market Insights (/market)
- Crop + location form
- AI price prediction
- Quick tips

### 8. Government Schemes (/schemes)
- Eligibility checker form
- AI recommendations
- Pre-loaded scheme cards with apply steps

### 9. Farming Calendar (/calendar)
- Interactive month calendar
- Crop + season form
- AI generated schedule

## Design Principles
- Green (#2E7D32) + White theme
- Real farmer imagery (no abstract)
- Simple, clean layout (no complex UI)
- Hinglish text throughout
- Mobile-first responsive
