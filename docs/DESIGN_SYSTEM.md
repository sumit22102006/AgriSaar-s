# 🎨 Design System — AgriSaar

## Colors

### Primary (Green)
| Token | Hex | Usage |
|-------|-----|-------|
| primary-50 | #e8f5e9 | Backgrounds, hover states |
| primary-100 | #c8e6c9 | Borders, dividers |
| primary-200 | #a5d6a7 | Secondary backgrounds |
| primary-400 | #66bb6a | Accents, highlights |
| primary-800 | #2e7d32 | **Main brand color** |
| primary-900 | #1b5e20 | Dark accents, text |
| primary-950 | #0d3b11 | Footer, dark sections |

### Earth Tones
| Token | Hex | Usage |
|-------|-----|-------|
| earth-100 | #f5e6d0 | Warm backgrounds |
| earth-400 | #c08a42 | Soil-related UI |

### Semantic
| Color | Hex | Usage |
|-------|-----|-------|
| Success | #22c55e | Good soil, high scores |
| Warning | #f59e0b | Medium scores, caution |
| Danger | #ef4444 | Low scores, errors |
| Info | #3b82f6 | Weather, tips |

## Typography
- **Font:** Outfit (Google Fonts)
- **Weights:** 300 (light), 400 (regular), 500 (medium), 600 (semibold), 700 (bold), 800 (extrabold)

## Spacing
- Cards: `p-6`, `rounded-2xl`
- Sections: `py-10` to `py-20`
- Max width: `max-w-7xl`

## Components
- `.card` — White card with green border shadow
- `.btn-primary` — Green button
- `.btn-secondary` — Outlined green button
- `.input-field` — Input with green focus ring
- `.glass-card` — Glassmorphism card
- `.gradient-bg` — Green gradient background

## Animations
- `animate-fade-in` — Opacity 0→1
- `animate-slide-up` — Translate + fade
- `animate-pulse-slow` — Slow pulse
- `animate-bounce-slow` — Slow bounce

## Icons
- Emojis for farming context (🌾🧪🌱🌧️📊💰)
- Lucide React for UI icons
